# django-course-exam-system

This is a full-stack Django application for managing online courses and exams.

## 🚀 Features

- Course management
- Lessons inside courses
- Questions and multiple choices
- Exam submission system
- Automatic result evaluation
- Bootstrap UI

## 🛠 Technologies

- Django
- HTML / CSS
- Bootstrap
